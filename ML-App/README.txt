This application predicts whether a patient is at risk of heart failure.

In order to run the application locate the folder ML-App using the command line. To run the application
type python (or python3) main.py. This will load a full screen GUI. Input values and press 'Predict Risk'.
This will initiate the training of a machine learning model. Note that this will only happen the first time
that the use presses this button, for future predictions the model is saved in the cache. A pop up box will
then alter the user whether the given patient information indicates a risk of heart failure.
